Ingeniería del software

El software es una reunión de aplicaciones que se instalan y ejecutan dentro de un sistema de computacional.

Definición

El software es una reunión de aplicaciones que se instalan y ejecutan dentro de un sistema de computacional. Estos programas los construyen los ingenieros de software. Se construye como cualquier otro producto, buscando principalmente la satisfacción del usuario o cliente, lo cual se refleja en la calidad con la cual es construido el producto. El software se ha convertido en un factor que limita la evolución de los sistemas informático
